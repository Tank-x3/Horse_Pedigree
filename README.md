# 5代血統表 作成ツール

競走馬の「5代血統表」を、誰でも簡単に作成・管理できるWebツールです。
実在の名馬はもちろん、ゲームや小説に登場する「架空馬（オリジナル馬）」の血統設定や、その共有にも最適です。

## 🌟 このツールの特徴

### 1. サクサク作れる、快適な入力
* **オートコンプリート:** 一度誰かが登録した馬は、みんなで共有。名前を入れるだけで親系統まで一発で呼び出せます。
* **自動補完機能:** 「母の名前は決めていないけど、母の父は設定したい」そんな時も大丈夫。空欄の部分をツールが自動で埋めて、家系図を繋げてくれます。
* **リセット機能:** 連続してたくさんの馬を作りたい時も、ボタン一つで次の準備が完了します。

### 2. 本格的な血統分析
* **クロス（インブリード）自動判定:** 5代以内の血統を自動的に分析し、発生しているクロス（例: サンデーサイレンス 3x4）を計算して表示します。
* **詳細データ:** 生年、毛色、生産国、ファミリーナンバー（F-No.）、系統名まで細かく記録・表示できます。

### 3. 作った血統表を画像で保存
* **画像出力:** 作成した血統表は、きれいな画像ファイル（PNG）としてダウンロードできます。SNSでの共有や、作品の資料として使えます。
* **シンプルモード:** 情報を絞ってすっきり見せたい時は「シンプル表示」に切り替え可能です。

### 4. データをクラウドで共有
* 入力したデータは「共有データベース」に保存されます。
* あなたが登録した馬は、他のユーザーも呼び出せるようになり、コミュニティ全体でデータベースが育っていきます。

### 5. ダークモード対応
* 画面右上のボタンで、目に優しい「ダークモード」に切り替えられます。（画像保存時は自動的に白背景になります）

## 🐎 使い方ガイド

### 基本的な流れ
1.  **「入力フォーム」**タブで、対象馬の名前や情報を入力します。
2.  父、母...と遡って入力していくと、右側の**「血統表プレビュー」**がリアルタイムに更新されます。
3.  登録済みの馬は、名前を入力し始めると候補が表示されます。選択すると親の代まで自動で埋まります。
4.  完成したら**「サーバーに保存」**を押してデータを登録・更新します。
5.  **「血統表を画像として保存」**ボタンで、画像をダウンロードできます。

### 架空馬（オリジナル馬）の登録
* 実在しない馬を登録する場合は、必ず**「架空馬」チェックボックス**をONにしてください。
* 架空馬は、詳細な情報（欧字名や生年など）がなくても登録可能です。

### データの修正について
* 一度登録した馬の情報を修正したい場合は、その馬を呼び出して内容を書き換え、「サーバーに保存」を押してください。
* **馬名自体の修正:** 誤字などで馬名そのものを直したい場合は、入力欄の下に出る「⚠️ 既存データの馬名を修正する」にチェックを入れてから書き換えてください。チェックが入っていない場合、安全のため自動的に別の馬として登録されます。

---

## ⚠️ ご利用上の注意
* このツールは**共有データベース**を使用しています。既存の実在馬データを意図的に改変したり、不適切なデータを登録したりすることはご遠慮ください。
* 実在馬を登録する際は、データの重複を防ぐため、可能な限り正確な「欧字馬名（英語名）」と「生年」の入力にご協力ください。
* 古い日本生産馬など、正確な欧字馬名が不明、または実際に登録されていない場合は、ローマ字表記で入力してください。
