# 競走馬血統表作成ツール
競走馬の血統表を作成するツールです。架空馬やオリウマ（オリジナルウマ娘）などにも使えます。  
2代血統表（祖父母の代）～5代血統表（父父父父父などの代）まで作成可能です。  
入力したデータをDB（CSV形式）として保存することで、DBに登録されている馬の情報を遡って祖の情報を自動入力することができます。  
また、CSVファイルを編集してツール外でDBに情報を追加することもできます。  
DBでは、ある馬の両親情報のみが1行に記録されており、ツールではある入力欄にDB登録済の馬をオートコンプリートで指定すると自動的にその親、親の親...と自動的に可能な限りデータを遡って呼び出すことでまとめて情報を呼び出すことが可能です。  
作成した血統表は画像として保存することができます。使用環境にかかわらず、世代数ごとに固定サイズで出力されるので共有して楽しむのにも便利です。
