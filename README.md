# 競走馬血統表作成ツール
競走馬の血統表を作成するツールです。架空馬やオリウマ（オリジナルウマ娘）などにも使えます。  
2代血統表（祖父母の代）～5代血統表（父父父父父などの代）まで作成可能です。  
このツールは現在、**【共通DB】**  を採用しています。  
入力フォームに馬名を入力する際、登録済の馬と一致する名前がある場合、候補が表示されます。  
表示された候補の中に自分が入力したい馬があった場合はクリック/タップすると、その馬の情報が入力中だった入力欄に補完され、  
更にその馬の親情報が登録されている場合は可能な限り遡って自動的に入力されます。  
入力したデータは「サーバーに保存」ボタンを押すと共通DBに追加され、次回以降はオートコンプリートで利用できるようになります。  
また、「共通DBにデータを追加したくない」「共通DBには追加してもいいけど自分でもローカルにバックアップが欲しい」という方向けに、「ローカルに保存」を押すことで現在の共通DBに登録されているデータも合わせた全データをローカルに保存することができます。
